Kiss 'em All!

Ein Spiel für die Ludum Dare 32. Das Team hatte zwei Tage Zeit, um das Spiel fertig zu stellen - Ein Menge Stress, Panik und ... Spaß! Dieses Spiel ist nicht wirklich fertig, die Entwicklung wurde nach dem Ende der Ludum Dare gestoppt. Die .jar ist auf Gamejolt erhältlich und der Quellcode kann auf Github eingesehen werden.

Pneumatic Man

Ein Spiel, das während der GameDevWeek entwickelt wurde. Ich hatte als Mitglied im Physics-Team jede Menge Spaß an der Entwicklung. Es ist ein simpler, aber spaßiger kleiner Patformer. Zum Spielen muss einfach die .jar runtergeladen und gestartet werden. Der Quellcode kann auf Github eingesehen werden.
